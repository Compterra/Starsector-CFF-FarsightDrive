# Farsight Drive

Current version: `4.2.9` for Starsector `0.98a`.

A restoration and English-polish fork of Farsight Drive, a secretive Eridani biotech faction built around living Reflecting-Light crystal ships, relief logistics, phasework, karma systems, and sealed corporate memory.

## Features

- Farsight Drive faction content with custom ships, fighters, weapons, systems, hullmods, markets, and lore text.
- Living-crystal mechanics, karma interactions, crystal infestation hooks, and FSD-only ascension equipment.
- The FSD Charter structure chain: Relief Charter and Reflecting-Light Liaison Spire, with AI core support, improvements, storefront access, and unique building art.
- English restoration pass for hulls, weapons, hullmods, colony structures, descriptions, and faction presentation.
- Compatibility and maintenance work for Starsector `0.98a`, including Version Checker support.

## Requirements

- Starsector `0.98a`
- LazyLib
- MagicLib
- GraphicsLib
- zz BoxUtil
